# raylib-games

A collection of small SNES-style games implemented in C using raylib.

## Games

### Excite Bike
A remake of the classic Excite Bike game.

**Build:**
cd excite-bike
./build.sh 

**Run:**
./excite-bike-c## 

Requirements
- raylib (install via Homebrew: `brew install raylib`)
- GCC compiler
- macOS (for the current build script)
